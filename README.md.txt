# 📡 AWS Static Website Hosting & Budget Alert Project

This project demonstrates core AWS Cloud Practitioner (CLF-C01) skills by deploying a public static website on Amazon S3, enabling logging, monitoring usage with CloudWatch, and setting up a cost budget alert using AWS Budgets.

---

## 🌐 Live Demo

✅ Hosted on: Amazon S3  
🔗 [Visit Live Site](http://your-bucket-name.s3-website-region.amazonaws.com)

---

## 🧱 AWS Services Used

| Service        | Purpose |
|----------------|---------|
| Amazon S3      | Host static HTML website |
| S3 Logging     | Track visitor access logs |
| CloudWatch     | Monitor usage metrics |
| IAM            | Manage bucket access policies |
| AWS Budgets    | Set cost alerts and manage free tier usage |

---

## 🚀 What I Did

- Created an S3 bucket for static website hosting
- Uploaded and served a basic `index.html`
- Enabled public read access with bucket policy
- Created a second bucket for server access logging
- Enabled logging to that target bucket
- Viewed S3 usage metrics in CloudWatch
- Created a monthly AWS Budget with 80% email alert threshold
